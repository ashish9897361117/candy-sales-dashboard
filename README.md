# 🍬 Nassau Candy Sales Analytics Dashboard

## 📊 Project Overview
An interactive business intelligence dashboard built using Streamlit to analyze sales performance, profitability, and cost efficiency.

## 🚀 Live Demo
👉 https://candy-sales-dashboard-1117.streamlit.app/

## 📌 Key Features
- Product profitability & margin analysis  
- Division performance comparison  
- Cost vs sales diagnostics  
- Profit concentration (Pareto analysis)  
- Interactive filters & modern UI  

## 🛠 Tech Stack
Python | Pandas | Plotly | Streamlit
